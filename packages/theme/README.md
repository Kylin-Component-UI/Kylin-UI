# Kylin-UI theme

## Instroduction

This is a plan for style customization of Kylin-UI.

## CSS Design

### How to use?

1. Common

```tsx
<Button type="primary" className="what you want"></Button>
```

2. customize

```tsx
<Button
	type="primary"
	className="what you want"
	bg="what you want"
></Button>
```

For detail, you can learn about unocss

## Install

```bash
  npm install @kylin-ui/theme
```
